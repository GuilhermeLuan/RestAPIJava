## Endpoints

Aqui estão os endpoints disponíveis nesta API:

### `GET /api/produtos`

Recupera uma lista de todos os produtos.

#### Parâmetros de consulta

- `categoria`: (opcional) Filtra os produtos por categoria.

#### Exemplo de resposta

```json
[
  {
    "id": 1,
    "nome": "Produto 1",
    "categoria": "Categoria 1"
  },
  {
    "id": 2,
    "nome": "Produto 2",
    "categoria": "Categoria 2"
  }
]
